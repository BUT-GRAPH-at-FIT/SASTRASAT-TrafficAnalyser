# TrafficAnalyser

This project relies strictly on Python 3.11 due to dependency on deprecated implementation from Tensorflow 1.x.
To run TrafficAnalyser, install requirements in the following order:

```shell
pip install --upgrade pip
pip install -r requirements-tf-libs.txt
pip install -r requirements.txt
```

*For running feature analysis using one of the Jupyter notebooks in `notebooks/` directory, you will need a separate environment due to requirements conflicts. The `requirements-notebooks.txt` contains all packages that are required to run the notebooks.*
